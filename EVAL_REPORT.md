## 检索策略

```
查询 → 意图分类（4 类意图）
     → 向量检索（Chroma + text-embedding-v3, weight=0.3~0.7）
     → BM25 关键词检索（jieba 分词, weight=0.3~0.7）
     → RRF 融合排序
     → 关键词 Rerank
     → Top-K 结果
```
